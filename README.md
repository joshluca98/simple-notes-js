# simple-notes-js-luca
Module 2 JS Project for Bay Valley Tech

A simple notes app that runs in the browser. Built using HTML, CSS, and JavaScript.

Functionality:
<ul>
  <li>Add Note</li>
  <li>Display Note</li>
  <li>Edit Note</li>    
  <li>Remove Note</li>
  <li>Search Notes</li>
  <li>Sort Notes</li>
  <li>Light/Dark Mode Switcher</li>    
</ul>
